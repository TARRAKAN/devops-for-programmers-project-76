1. To install the ansible roles use:
```commandline
make install_roles
```
2. To prepare the environment use:
```commandline
make prepare
```
3. To deploy the app use (It makes all the previous steps automatically):
```commandline
make deploy
```
4. To stop the app use:
```commandline
make stop
```
